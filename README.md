<h1>📘 JavaScript Assignment 3 – Basic HTML & JS Integration</h1>
<h3>📖 Description:</h3>
This project is a simple practice assignment that demonstrates the basic structure of an HTML document and<br> how to connect an external JavaScript file to a webpage.
The project includes a basic HTML boilerplate setup and links a JavaScript file using the <script> tag. It is designed to help beginners understand how HTML and JavaScript<br> work together in a web page.

When the page loads, it displays a heading message and executes the linked JavaScript file.
<hr>
<h3>🚀 Features:</h3>
<li>Basic HTML5 Boilerplate Structure</li>
<li>Clean and simple webpage layout</li>
<li>Demonstrates how to link an external JavaScript file</li>
<li>Beginner-friendly JavaScript practice project</li>
<li>Organized and readable code</li>
